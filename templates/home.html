Данное web-приложение разработано для автоматического определения ключевых слов и аннотирования текстовой информации из открытых источников для решения задачи информационного поиска. Данное приложение на работу с научными статьями.

Раздел "Поиск" предназначен для поиск в различных источника. Существет два основных режима: "Библиотека" и "ВКонтакте". Режим "Библиотека" предназначен для поиска в общей или личной библиотеках, поиск возможен по автору, ключевым словам и заголову. Режим "ВКонтакте" предназначен для поиска в добавленных группах, поиск возможен по всем группам и по определенной группе(необходимо через запятую указать сначала название группы и далее слово, по которому производиться поиск,нр "аналитика, когортный анализ") После выбора необходимых настроек и после ввода слова необходимо нажать на кнопку "Поиск". Далее будет показан результат поиска, при необходимости можно вернуться на исходную страницу нажав на "Назад".

Раздел "Загрузить" предназначен для загрузки метаданных о статьях в личную библиотеку, чтобы в дальнейшем в ней можно было совершать поиск. Правила загрузки: файл должен быть формата csv или xlsx c разделителем ';', первый столбец - автор, далее название и ключевые слова. Для загрузки необхоимо нажать "Выберете файл" и после "Загрузить". Далее Вы окажитесь на странице для редакции. Для загрузки в базу данных нажмите на "Сохранить в БД".

Раздел "Найти похожие" предназначен для аннотирования и выделения ключевых слов в одиночных текстах, а также для поиска похожих тектов. Необходимо вставить текст в первое поле ввода, далее либо самостоятельно заполнить поле "Клчючевые слова" и "Аннотация", либо создать их автоматически, нажав на кнопку "Создать". Режим мера сходства влияет на автоматическое аннотирование. Далее для поиска похожих статей необходимо выбрать место поиска, личная или общая библиотека, и нажать на "Найти похожие". Ниже появиться список 5ти похожих текстов.

Раздел "Личная библиотека" предназначен для просмотра данных личной библиотеки. Можно удалить ненужные статьи, для этого напротив соответсвующей записи нажмите "Удалить".

Раздел "Список групп" предназначен для просмотреа, удаления и обновления добавленных групп из "Вконтакте"

Раздел "Добавление группы" предназначен для добавления групп из "Вконтакте". Для этого в пле ввода необходимо ввести id группы и нажать на 'Добавить'.

Раздел "Пакетная обработка" предназначен для аннотирования и выделения ключевых слов большого количества текстов. Правила загрузки: файл должен быть формата csv или xlsx c разделителем ';', первый столбец - автор, далее название и текст. Для загрузки необхоимо нажать "Выберете файл" и после "Загрузить". Далее Вы окажитесь на странице для редакции. Для сохранения на ПК нажимите "Сохранить".
